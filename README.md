# **Supermarket Comparison Web App**

Compare **supermarket prices**, **deals**, and **availability** across stores in **Kenya** (and beyond). Live demo: **https://gikenn.github.io/supermarket-comparison-web-app**

![Screenshot or logo placeholder]

## ✨ **Features**
- ✅ **Real-time price comparison** across multiple supermarkets
- 📱 **Responsive design** - works on mobile/desktop
- 🛒 **Product search & filtering** by category/brand
- 📊 **Price history charts** & deal alerts
- 🌍 **Location-based store finder** (Machakos County & nationwide)

  
## 🔧 **Deployment**
1. Fork/Clone this repo
2. Push to your **gh-pages** or **main** branch
3. GitHub **Settings > Pages > Deploy from branch: main**
4. Live at `yourusername.github.io/supermarket-comparison-web-app`
